# 🚀 Automatisation avec Python

Ce projet est une **application d'automatisation** développée en Python. Il permet d'interagir avec l'API Spotify, d'authentifier l'utilisateur, d'ouvrir une interface graphique, et de contrôler la lecture de musique. Ce projet met en évidence mes compétences en **développement Python**, **gestion de projet**, **interface utilisateur**, et **intégration d'APIs**.
